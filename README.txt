Pre-requisites:
---------------
- Install nltk, pyenchant and autocorrect.
    - pip install nltk
    - pip install pyenchant
    - pip install autocorrect
- Stanford NER
    - Download jar files from http://nlp.stanford.edu/software/stanford-ner-2014-06-16.zip

Input Data Format:
------------------
- An input is a single sentence.
